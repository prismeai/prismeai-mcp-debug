# Legacy Product Documentation

This directory contains documentation for the legacy product architecture: AI Store, AI Knowledge, AI Knowledge Client, SecureChat, AI Governance, AI Insights, and AI Collection.

For one-product product scopes, use:

| Legacy concept | Current documentation |
|----------------|-----------------------|
| AI Store agent marketplace | `product-agent-factory` |
| AI Knowledge project/agent runtime | `product-agent-factory` |
| AI Knowledge documents/RAG/vector store | `product-storage` |
| Knowledge Client query | Agent Factory `messages/send` or `messages/stream` |
| AI Knowledge model execution | `product-llm-gateway` |
| AI Knowledge tools | `product-capabilities` and `capability-workspaces` |
| Legacy Governance | `product-governance-v2` |
| Legacy Insights | `product-insights-v2` |
| Legacy Collection | `product-collection-v3` |
